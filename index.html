::: HOW IT WORKS

1. Feature Encoding: Each feature is a unique bit. If you have 3 features:

  • Feature A = 1 (2⁰)
  • Feature B = 2 (2¹)
  • Feature C = 4 (2²)

2. State Calculation: Your current state is the sum of bits. If you have eaten A and C, your state is 5 (1 + 4).

3. Monotonic Transition: Transitions use the OR (|) operator. You can only add features, never lose them. Once a bit is 1, it stays 1.

NextState = CurrentState | SymbolMask
